> Skills are Claude's internal how-tos — concrete procedures for recurring tasks (debugging, writing plans, shipping PRs). Claude triggers them automatically when a matching situation appears. The value is the small set you actually use, not the full catalog you installed.

---

`~/.claude/skills/` contains a large catalog of skills plus a few standalone `.md` helpers.
Skills are invoked via `/skill-name` in any Claude Code session.
Many belong to the **gstack** suite — preamble-based, auto-suggested on matching phrases.

## What it is

SkillSwitch (`skillctl`) is a command-line tool for managing AI skills across any AI coding assistant. Install skills, toggle them on/off, create profiles for different contexts (solo work, client work, deep focus), and generate a catalog of everything available.

Install: `npm install -g skillswitch`

## The problem

As AI assistants gain skill ecosystems, managing which skills are active across different tools and workflows becomes unwieldy. SkillSwitch treats skills like packages — installable, versioned, toggleable, and portable.

## Supported platforms

Claude Code, Gemini CLI, Codex CLI, Aider, Amp, Factory Droid.

## Current status

v0.2.0. Published to npm as `skillswitch`. Open-source, MIT license.

## Stack

TypeScript, ESM, Node.js, commander.
